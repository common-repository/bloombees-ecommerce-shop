== Description ==

Bloombees offers a unique and innovative cross-device platform, empowering small merchants to sell worldwide on their social networks in just a few minutes. We provide a simple and effective way to share products on social media and facilitate instant transactions in just about any currency and payment method, turning fans and followers into potential customers and transforming “likes” into “buys”. Welcome to Instant Commerce.

WP plugin which adds shortcode for inserting bloombees catalog into page content.

== Usage ==
~~~
[bb_catalog brand="your brand name"]
~~~

Options:

brand - optional, brand name to load products, if not specified default value from plugin settings will be used

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/bloombees-ecommerce-shop directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings->Bloombees Catalog Settings and enter your Bloombees brand name. If you don't have a brand name yet please create an account at bloombees.com and upload your products first.
4. In Wordpress create a New Page and as content copy and paste this: [bb_catalog brand="your brand name"]
5. We recommend to use a page that has a Full Width layout.
6. Save and publish your new page (you might have to add it to your navigation menus)
7. Your products are now visible on your WP site, congrats!
